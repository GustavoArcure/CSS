Alinhamento de elementos e conteúdo usando Flexbox

Flexbox é um mecanismo do CSS3 que permite facilmente alinhamento e distribuição de elementos de conteúdo no HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats, positions na resolução de dificuldades na construção de layouts.

As propriedades do flexbox são os seguintes:

  • flex-direction: Define qual direção o container deseja empilhar os itens (column, column-reverse, row, rou-reverse).
  • flex-wrap: Especifica se os itens flexíveis devem ser encapsulados ou não (flex-wrap: wrap, nowrap, wrap-reverse).
  • flex-flw: Propriedade abreviada para definir flex-direction e flex-wrap (flex-flow: row wrap).
  • justify-content: Alinhamento de itens (justify-content: center, flex-start- flex-end, space-around, space-between).
  • align-items: Propriedade é usada para alinhar os itens flexíveis.
  • align-content: Propriedade é usada para alinhas as linhas flexíveis.
